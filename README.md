# Diabetes Prediction System 🩺

This beginner-friendly Machine Learning project predicts whether a person is diabetic based on health attributes.

## Features
- Data preprocessing
- Logistic Regression model
- Accuracy evaluation

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn

## Dataset
https://www.kaggle.com/datasets/mathchi/diabetes-data-set

## How to Run
pip install -r requirements.txt
python app.py
