# Diabetes Prediction Using Machine Learning

## Overview

This project uses Machine Learning techniques to predict whether a person is likely to have diabetes based on various health parameters. The objective is to assist in early detection and support data-driven healthcare decisions.

## Problem Statement

Diabetes is one of the most common chronic diseases worldwide. Early prediction can help patients seek timely medical attention and adopt preventive measures. This project aims to build a predictive model using patient health data.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Dataset Features

The dataset contains several medical predictor variables such as:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age

## Project Workflow

### 1. Data Collection

Imported the diabetes dataset for analysis and prediction.

### 2. Data Preprocessing

* Checked for missing values
* Cleaned and prepared data
* Performed feature selection

### 3. Exploratory Data Analysis (EDA)

* Analyzed patient health indicators
* Visualized feature distributions
* Identified relationships between variables

### 4. Model Development

* Applied Machine Learning algorithms
* Trained classification models
* Evaluated model performance

### 5. Model Evaluation

Performance measured using:

* Accuracy Score
* Precision
* Recall
* F1 Score
* Confusion Matrix

## Results

The model successfully predicts diabetes risk based on patient health parameters and demonstrates the practical application of Machine Learning in healthcare analytics.
